missionbio.mosaic.cnv.Cnv.run_umap¶

Cnv.run_umap(layer=None, attribute=None, output_attr='umap', **kwargs)¶

Perform UMAP on the given data.

Adds output_attr to the row attributes.

Parameters
layerstr

The layer to be used for UMAP.

attributestr

The row attribute to be used for UMAP.

output_attrstr

The name of the row attribute where the UMAP output is.

kwargsdictionary

Passed to UMAP.

Raises

When both layer and attribute are provided. Only one is permitted at a time.
