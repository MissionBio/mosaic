missionbio.mosaic.protein.Protein.scatterplot¶

Protein.scatterplot(attribute, labels=False, feature_ids=None, layer=None, title='')¶

Scatter plot for all barcodes.

Requires a 2-D row attribute.

If labels is False, feature_ids are not given, and the layer is None, then this plot can be used to create custom clusters using the plotly lasso tool.

Not selecting any cell deselects all clusters, and double clicking refocuses all points.

Parameters
attributestr

The row attribute to use as coordinates, the most likely value is ‘umap’. This row attribute must be 2D.

labelsbool

Whether the plot should be colored by the labels that have been set.

feature_idslist

The features to color the plot by.

layerstr

The layer to use for the color when the feature is provided.

titlestr

The title given to the plot after the name of the assay.

Returns
figplotly.graph_objs.FigureWidget
Raises
ValueError

Raised in the following cases. When the attribute is not present in the assay. When both labels and feature_ids are provided as arguments. When feature_ids are provided but a corresponding layer is not provided.
