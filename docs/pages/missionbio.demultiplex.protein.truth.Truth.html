Truth#

missionbio.demultiplex.protein.truth.Truth

class Truth(celltypes: Sequence[CellType] = (), version: str = 'NA')#

A phylogenetic tree of cell types.

Parameters:

  • celltypes – A list of cell types.

  • version – The version of the truth.
