SampleGroup.merge

SampleGroup.merge#

SampleGroup.merge() → missionbio.mosaic.sample.Sample#

Merge all the samples into on Sample object.

A sample group contains a list of samples.

>>> import missionbio.mosaic as ms
>>> grp = ms.load_example_dataset("Multisample PBMC")
>>> [s.name for s in grp]  # Group of samples
['Sample 1', 'Sample 2']

For various analyses data from the samples has to be concatenated. This can be done using the merge command to obtain the Sample object on which further analysis can be performed.

>>> sample = grp.merge()
>>> sample.name  # A single sample with all the cells
'Sample 1, Sample 2'
Returns
Sample with all the assays merged
