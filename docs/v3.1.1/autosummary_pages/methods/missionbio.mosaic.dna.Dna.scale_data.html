Dna.scale_data#

missionbio.mosaic.dna.Dna.scale_data

Dna.scale_data(layer, output_label='scaled_counts')#

Z-score normalization

Ensures that all id have a mean of 0 and a standard deviation of 1.

Adds scaled_counts to the layers.

Parameters:
layerstr / np.array

The data to scale. Uses get_attribute() to retrieve the value constrained by row+col.

output_labelstr, default SCALED_LABEL

The name of the output layer.
